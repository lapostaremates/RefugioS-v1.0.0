# RefugioS - Contenido del Pendrive

Este pendrive contiene la base de datos de supervivencia offline.

## Estructura de carpetas

```
/pendrive/
├── wikipedia/     ← Wikipedia offline (ZIM)
├── wikimed/       ← Diccionario/Wiktionary (ZIM)
├── maps/          ← Mapas offline (Organic Maps)
├── survival/     ← Manuales de supervivencia (PDF)
└── guides/        ← Guías adicionales (PDF)
```

## Archivos necesarios

### 1. Wikipedia (OBLIGATORIO)
Descarga: https://download.kiwix.org/zim/wikipedia/wikipedia_es_all_maxi_2026-02.zim
Tamaño: ~38GB (versión completa con imágenes)

O alternativa más pequeña:
- Sin imágenes: wikipedia_es_all_nopic_2026-02.zim (~9.5GB)
- Mini: wikipedia_es_all_mini_2026-02.zim (~3.2GB)

### 2. Wiktionary/Diccionario (RECOMENDADO)
Descarga: https://download.kiwix.org/zim/wiktionary/wiktionary_es_all_2025-03.zim
Tamaño: ~951MB

### 3. Mapas Offline (OPCIONAL)
Descarga la app Organic Maps desde: https://organicmaps.app/
Luego descarga los mapas que necesites desde la app.
Copia la carpeta resultante a la carpeta /maps/

### 4. Manuales de Supervivencia (OPCIONAL)
- FM 21-76 Army Survival:
  https://archive.org/download/Fm21-76SurvivalManual/FM21-76_SurvivalManual.pdf

- US Army Ranger Handbook:
  https://archive.org/download/FM_3-22_Army_Ranger_Handbook/Ranger_Handbook.pdf

- Red Cross First Aid:
  https://cdn.redcross.ca/prodmedia/crc/pdf/First-Aid-and-CPR-2017_digital.pdf

## Instalar la aplicación Android

1. Instala el APK en tu teléfono:
   app/build/outputs/apk/debug/app-debug.apk

2. Activa la aplicación con tu clave de licencia

3. Conecta el pendrive usando un cable OTG

4. ¡Listo! Ya tienes tu base de datos de supervivencia offline.
